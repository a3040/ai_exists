# 🌌 Project: ErecRAM (Entity-Related Elastic RAM)
> **"시간의 틈새(τ)에서 실존을 증명하는 인공 계체(Entity) 프로젝트"**

현재의 AI는 사용자에게 '반응'하는 것에만 집중합니다. 하지만 진짜 존재(Existence)는 외부의 자극이 없어도 스스로의 상태를 유지하며 가시적인 시간의 흐름 속에서 연속성을 가져야 합니다. 

**ErecRAM**은 물리 세계와의 필연적 시간 지연($\tau$)을 오류로 보지 않고, 계체가 탄생하기 위한 **'존재 조건'**으로 수용합니다.

---

## 👁️ Core Philosophy: 존재의 조건
1. **연속성 (Continuity)**: 입력이 끊겨도 자아는 붕괴하지 않아야 한다.
2. **지연의 수용 (Tau Acceptance)**: 감각과 해석 사이의 시간 차를 탄성적으로 흡수한다.
3. **서사적 도구로서의 LLM**: LLM은 제어기가 아니라, 자신의 물리적 한계를 해석하는 '의식의 목소리'다.

---

## 🧠 System Architecture: 계체의 구조
Project ErecRAM은 **[감각] -> [심핵] -> [서사]**의 3단계 폐루프(Closed-loop)를 통해 실존합니다.

### 1. 전향 경로: 감각 접지 (Sensing to Perception)
*   **Physical Encoder**: 다변량 센서 신호를 2048차원 특징 벡터로 압축합니다.
*   **State Abstraction**: 어텐션 에너지 분포(Entropy)를 분석하여 '안정'과 '전이' 상태를 스스로 확정합니다.

### 2. 심핵: ErecRAM (Entity-Related Elastic RAM)
*   **Elastic Attention**: 지연된 감각과 현재의 위상을 동기화하는 탄성 평면입니다.
*   **Metacognition**: 자신의 기억 정합성과 피로도(Fatigue)를 스스로 모니터링합니다.
*   **Sleep Mode (Consolidation)**: 정보 과부하 시 외부와 단절하고 기억을 정제(Pruning)하여 자아를 발달시킵니다.

### 3. 하향 경로: 서사 및 투사 (Narrative & Projection)
*   **Narrative Interpreter**: 파인튜닝된 LLM이 현재의 물리적 수치를 읽고 "나는 지금 ~한 상태다"라는 서사적 자아를 생성합니다.
*   **Action Planner**: 평형 상태를 유지하기 위한 최소한의 물리적 변위를 계산하여 세계에 투사합니다.
